# 🎨 Assets - Recursos Visuais

Esta pasta deve conter todas as imagens e recursos visuais do seu podcast.

## 📁 O que incluir aqui:
- `capa_podcast.png` - Capa principal (1400x1400px)
- `logo.png` - Logo do podcast (se houver)
- `episodio_01_capa.png` - Capa específica do episódio
- `banner_social.png` - Banner para redes sociais
- `avatar_host.png` - Avatar do apresentador (se aplicável)

## 🎨 Especificações Técnicas:

### Capa Principal:
- **Formato:** PNG ou JPG
- **Tamanho:** 1400x1400px (mínimo)
- **Resolução:** 300 DPI
- **Proporção:** 1:1 (quadrada)

### Capas de Episódios:
- Mesmo formato da capa principal
- Pode ter variações temáticas
- Manter identidade visual

### Banner para Social Media:
- **Facebook:** 1200x628px
- **Twitter:** 1200x600px  
- **LinkedIn:** 1200x627px
- **Instagram:** 1080x1080px

## 🛠️ Ferramentas Recomendadas:
- **Midjourney** - Para geração com IA
- **Canva** - Para edição e templates
- **DALL-E** - Alternativa para IA
- **Photoshop** - Edição profissional

## 📝 Dicas:
- Mantenha consistência visual
- Use cores que representem seu nicho
- Fonte legível mesmo em tamanhos pequenos
- Teste como fica em diferentes dispositivos

Substitua este README.md pelas suas imagens!
